# music_box
用python实现下载付费音乐的demo

## Installation
```
git clone https://github.com/ChenJunhan/music_box.git 
git clone git@github.com:ChenJunhan/music_box.git
```

## Usage
1.安装pip

2.在项目文件夹中执行
`pip install -r requirements.txt`

3.安装完所需要的库后执行即可搜索音乐
`python music_box.py`
